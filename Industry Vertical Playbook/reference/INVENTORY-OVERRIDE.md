# Inventory Positioning — Source of Truth Override

**Status:** Method has NOT launched native inventory management. This is several quarters away.

**The method-info skill is WRONG on this topic.** It says "Method now has native multi-warehouse inventory management (launched Jan 2026, QBO only)." This is incorrect. Always use the positioning below instead.

## Correct Inventory Positioning

Method does NOT have native inventory management. The inventory story today is **QB sync plus customization**:

- Method surfaces QuickBooks' built-in inventory features (Inventory Parts for QBO + Desktop, Assembly Items and Advanced Inventory for Desktop only) inside CRM and operational workflows.
- For complex inventory needs beyond QB's native capabilities, two proven customer paths exist:
  1. **Third-party API integrations** (e.g., SOS Inventory, Fishbowl)
  2. **Custom builds inside Method** for warehouse tracking, serial numbers, bin locations — real customer implementations, not theoretical

## What Method Is NOT

- Not a standalone inventory management system
- No native BOM, kitting, lot tracking, or shop floor control
- No native e-commerce integration
- Heavy manufacturers need a dedicated inventory tool alongside Method

## How to Talk About It

- Lead with honesty: "Method extends QB's inventory data into your operational workflows"
- Two proven paths for complex needs — always mention both
- Reference real customer examples (Duraline, Vintage Makers, Europena)
- Never claim native inventory features that don't exist
